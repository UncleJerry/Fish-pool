# Fish-pool

A backend project for [this iOS project](https://github.com/UncleJerry/Dailylife-with-a-fish)

## Features
- Signup, Signin & logout
- User Profile
- Notification
- Girlfriend/Boyfriend Account Matching

## Environment
- PostgreSQL 9 or above
- Node.js 6 or above
- NPM Package noted in package.json
